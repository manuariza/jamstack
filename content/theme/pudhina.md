# Pudhina

A simple, minimal Jekyll theme for a personal web page and blog, focusing on white space and readability

## Features

* Simple, minimal and clean
* Blog posts categorised
* Responsive
* Syntax Highlighter (using highlight.js).
* Font-Awesome Icons.
* Glitchy 404 page
